## Definition

```agda
_choose-ℕ_ : ℕ → ℕ → ℕ
0 choose-ℕ 0 = 1
0 choose-ℕ succ-ℕ k = 0
(succ-ℕ n) choose-ℕ 0 = 1
(succ-ℕ n) choose-ℕ (succ-ℕ k) = add-ℕ (n choose-ℕ k) (n choose-ℕ (succ-ℕ k))
```

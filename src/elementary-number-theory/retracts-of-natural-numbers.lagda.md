## Idea

If `i : A → ℕ` has a retraction, then `i` is a decidable map.

```agda
is-decidable-map-retr-ℕ :
  {l1 : Level} {A : UU l1} (i : A → ℕ) → retr i → is-decidable-map i
is-decidable-map-retr-ℕ =
  is-decidable-map-retr has-decidable-equality-ℕ
```
